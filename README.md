# Organizing Medical Transcriptions with the OpenAI API
This project aims to extract patient details from medical transcriptions and generate recommended treatments & ICD-10 codes using the OpenAI API.

## Files
- [notebook.ipynb](notebook.ipynb) — code for this project.
- [data/transcriptions.csv](data/transcriptions.csv) — dummy dataset medical transcriptions.

## Requirements
- Python 3.8+
- OpenAI API key available in the development environment.

## Installing required dependencies
```
pip install pandas openai
```
